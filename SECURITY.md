# Security Policy

## Reporting a Vulnerability
- Email: gkigkispetros (at) gmail.com
- Please include:
  - Affected component/version/commit
  - Reproduction steps or PoC
  - Expected impact and any mitigation ideas

## Handling
- This is a best-effort project; we will try to reproduce, assess impact, and propose a fix/mitigation when feasible.
- Coordinated disclosure is appreciated; we will work with you on a reasonable disclosure timeline.
